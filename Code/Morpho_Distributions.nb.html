Morphological Distributions by Species

Overview

This script fits a series of Bayesian distributional regression models to estimate the distribution of each skeletal measurement across four focal Florida rodent species. One model is fit per variable in var_accurate.

Model structure varies by variable type:

Size-controlled variables (Ppl, Hl, Rl, Uol, Fl, Ul): - mu ~ 0 + Genus_species + log_Mass - sigma ~ 0 + Genus_species - Missing log_Mass values are filled in beforehand using a standalone mass prediction model fit to the full dataset

Non-size-controlled variables (HPI, PES, MANUS, OLI): - mu ~ Genus_species - sigma ~ Genus_species

Predictions use species mean log_Mass for the size-controlled variables, so intervals reflect morphological variation at a typical body size for each species rather than being confounded by mass differences.

2. Fit Models

2a.

Standalone mass prediction model

Before fitting the size-controlled morphology models, we need to fill in missing mass values for focal specimens. A standalone regression model is fit to the full dataset (all species) to predict log_Mass from skeletal measurements. Posterior means are then used to impute missing values in dat_mice — simpler than the full mi() joint model and sufficient here since we only need a point estimate per specimen.

# Fit mass model to full dataset so all species inform the regression
# Predictors are the same complete skeletal measurements used elsewhere

mass_mod <- brm(
  mass_g ~ Hl + Hsw + Hdw + Uol + Ul + Rl + Fl + Fsw + Fdw,

  data    = dat,       # full dataset, not just focal species
 # prior   = p_mass,
  cores   = 4,
  seed    = 145,
  refresh = 0,
  silent  = 2

3. Extract Posterior Draws

With 0 + Genus_species parameterization, each species has its own named coefficient (b_Genus_speciesOchrotomys_nuttalli, etc.). We extract these directly using spread_draws() — no prediction step needed. This gives the posterior distribution of the species mean for each variable, already size-adjusted for the size-controlled variables.

The species name cleanup (removing “b_Genus_species” prefix) is done with str_remove() to recover clean Genus_species labels for plotting.

3a. Non-size-controlled variables

post_no_size <- map_dfr(names(dist_mods_no_size), function(var_name) {
  
  mod <- dist_mods_no_size[[var_name]]
  if (is.null(mod)) return(NULL)
  
  tryCatch({
    mod %>%
      spread_draws(`b_Genus_species.*`, regex = TRUE) %>%
      pivot_longer(
        starts_with("b_Genus_species"),
        names_to  = "Genus_species",
        values_to = "value"
      ) %>%
      mutate(
        Genus_species = str_remove(Genus_species, "b_Genus_species"),
        var           = var_name
      ) %>%
      select(.draw, Genus_species, value, var)
  }, error = function(e) {
    warning(glue("Draw extraction failed for {var_name}: {e$message}"))
    return(NULL)
  })
})

3b. Size-controlled variables

For these models the b_Genus_species coefficients are the species means at log_Mass = 0 (the grand mean on the z-score scale). Because all species are evaluated at the same mass value, size is held constant and comparisons reflect shape differences only.

3b. Size-controlled variables

For these models we predict at each individual’s own log_Mass, then average draws within species. This marginalizes over the observed mass distribution of each species rather than conditioning on a single reference value — so the species-level posterior reflects what you’d expect for a typical individual of that species at its actual size.

ppred_size <- map_dfr(names(dist_mods_size), function(var_name) {
  
  mod <- dist_mods_size[[var_name]]
  if (is.null(mod)) return(NULL)
  
  tryCatch({
    mod %>%
      predicted_draws(
        newdata = dat_mice %>%
          select(Genus_species, catalog, log_Mass) %>%
          filter(!is.na(log_Mass))
      ) %>%
      as.data.frame() %>%
      group_by(Genus_species, .draw) %>%
      summarise(value = mean(.prediction, na.rm = TRUE), .groups = "drop") %>%
      mutate(var = var_name)
  }, error = function(e) {
    warning(glue("Prediction failed for {var_name}: {e$message}"))
    return(NULL)
  })
})

# Plot with raw jittered points overlaid
# Both stat_pointinterval and geom_point use the same dodge.width = 0.5
# to ensure species are aligned across layers
morpho_plot <- function(groups) {
  
  ppred_plot <- ppred_all %>%
    filter(Genus_species %in% groups) %>%
    mutate(Genus_species = gsub("_", " ", as.character(Genus_species)))
  
  raw_plot <- dat_long %>%
    filter(gsub(" ", "_", Genus_species) %in% groups)
  
  ppred_plot %>%
    ggplot(aes(
      x     = fct_relevel(var, var_acc_dist),
      y     = value,
      shape = Genus_species
    )) +
    geom_rect(
      inherit.aes = FALSE,
      data        = shades,
      mapping     = aes(xmin = xmin, xmax = xmax, ymin = ymin, ymax = ymax),
      alpha       = 0.125
    ) +
    stat_pointinterval(
      .width    = 0.8,
      color     = "grey15",
      linewidth = 1.7,
      position  = position_dodge(width = 0.5)
    ) +
    geom_point(
      data     = raw_plot,
      mapping  = aes(x = fct_relevel(var, var_acc_dist), y = value,
                     shape = Genus_species),
      position = position_jitterdodge(dodge.width = 0.5, jitter.width = 0.1,
                                      jitter.height = 0),
      size     = 1.5,
      alpha    = 0.5,
      color    = "grey30"
    ) +
    scale_shape_manual(values = c(1, 15, 17, 19, 8)) +
    labs(x = "", y = "Z-score") +
    theme_pander(gM = FALSE) +
    theme(
      axis.text.x  = element_text(angle = 45, hjust = 1),
      panel.border = element_rect(colour = "grey40", fill = NA, linewidth = 0.7),
      legend.text  = element_text(face = "italic"),
      legend.title = element_blank()
    ) +
    guides(shape = guide_legend(override.aes = list(size = 4)))
}
